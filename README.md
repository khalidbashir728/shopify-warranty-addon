# shopify-warranty-addon
A production-ready, open-source Shopify warranty add-on system that automatically attaches warranty products to the cart, keeps quantities synchronized, and integrates seamlessly with Dawn, Horizon, and other Online Store 2.0 themes. Automatically add warranty products to Shopify carts with quantity sync, Ajax Cart support, Product Metafields, and Horizon/Dawn theme compatibility. Shopify Warranty Add-on System built with Liquid and JavaScript. Supports Product Metafields, Ajax Cart API, quantity sync, cart drawers, Dawn, and Horizon themes.
